하체 강화 루틴

🏋️ 루틴 목적: 근력 향상 / ⏱ 총 소요 시간: 약 50분 / 🧩 구성: 5개 운동 × 3세트

스쿼트 (Squat)

카테고리: 하체 / 부위: 대퇴사두, 둔근 / 장비: 없음 / 피로도: 2
세트: 3 / 반복: 20회 / 중량: 체중
다리를 어깨 너비로 벌리고, 등을 곧게 편 상태에서 엉덩이를 뒤로 빼며 앉았다 일어납니다.
무릎이 발끝을 넘지 않도록 주의합니다.

런지 (Lunge)

카테고리: 하체 / 부위: 햄스트링, 둔근 / 장비: 없음 / 피로도: 2
세트: 3 / 반복: 12회 (각 다리) / 중량: 체중
한쪽 다리를 앞으로 내디디며 무릎을 굽혀 앉았다가 다시 일어납니다.
허리를 곧게 유지하며 균형을 잡는 것이 중요합니다.

레그 프레스 (Leg Press)

카테고리: 하체 / 부위: 대퇴사두 / 장비: 레그프레스 기구 / 피로도: 3
세트: 3 / 반복: 15회 / 중량: 40kg
기구에 앉아 다리를 어깨 너비로 벌리고, 무릎을 굽혔다가 펴며 다리로 무게를 밀어냅니다.
허리가 들리지 않도록 고정합니다.

레그 컬 (Leg Curl)

카테고리: 하체 / 부위: 햄스트링 / 장비: 레그컬 머신 / 피로도: 2
세트: 3 / 반복: 15회 / 중량: 25kg
기구에 앉거나 엎드려서 다리를 굽혀 햄스트링을 수축시킵니다.
반동 없이 천천히 움직이는 것이 중요합니다.

카프 레이즈 (Calf Raise)

카테고리: 하체 / 부위: 종아리 / 장비: 덤벨 / 피로도: 1
세트: 3 / 반복: 20회 / 중량: 10kg
양손에 덤벨을 들고, 발뒤꿈치를 들어 종아리를 수축합니다.
천천히 내려오며 반복합니다.
